# Кворки SHTQ.PRO для KWORK

Позиционирование (из обложки): **проектирую сайты как систему. Структура. Логика. Дизайн.**
Тон: прямой, человеческий, без AI-воды («качественно», «в кратчайшие сроки», «уникальный дизайн» — не использовать). Без восклицательных знаков.

---

## Стратегия: какие кворки заводить

KWORK ищет по категории + ключевым словам. Один кворк = один запрос покупателя. Поэтому **не один кворк «сделаю сайт», а отдельный кворк под каждый стек и каждую услугу** — так ты занимаешь больше поисковой выдачи и каждый кворк точно отвечает на конкретный запрос.

### Приоритет запуска

**Tier 1 — запускай первыми (высокий спрос, быстрый старт):**
1. Лендинг на Tilda (Zero Block)
2. Лендинг на Next.js / React
3. Сайт на WordPress (ACF, Classic, без Gutenberg)
4. Дизайн сайта в Figma (UI/UX с нуля)

**Tier 2 — добавляй вторым заходом:**
5. Многостраничный сайт на Tilda
6. Сайт на Next.js + CMS (Sanity)
7. Вёрстка по макету Figma
8. Редизайн сайта

**Tier 3 — для потока отзывов и быстрых заказов:**
9. Статичный лендинг HTML/CSS/JS
10. Доработка и правки сайта

> Логика тиров: сначала самые частые запросы и то, что приносит первые отзывы. Дизайн и вёрстка по Figma подключают второй сегмент аудитории (у кого уже есть макет / нужен только дизайн). Доработки — дешёвый вход, быстро набивает рейтинг.
